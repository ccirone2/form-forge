How It Works

Every form needs two files: a schema (.json) that defines the form fields, and a template (.py) that generates the DOCX output. The schema drives the UI; the template drives the document. No data ever leaves the browser. 

your-repo/                         ← or just two local files
├── schemas/
│   └── my-form.json               ← defines fields, sections, layout
└── templates/
    └── my-form.py                 ← Python script with generate_docx(data)
