Connect a GitHub Repo

Point FormForge at a GitHub repository containing your form schemas and DOCX templates. It will fetch them and let you pick which form to fill out.

Repository

▸ optional: add personal access token (for private repos)
Token is only used in-browser and never stored or transmitted elsewhere. Needs repo scope for private repos.

Expected Repo Structure

your-repo/
├── schemas/
│   ├── onboarding.json        ← form schema (fields, sections, metadata)
│   ├── expense-report.json
│   └── incident-report.json
└── templates/
    ├── onboarding.py          ← python-docx generation script
    ├── expense-report.py
    └── incident-report.py

Each schema JSON references its template by name.
Templates are Python scripts exporting a generate_docx(data) function.
